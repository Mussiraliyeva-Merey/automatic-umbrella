# Экологические привычки будущего

Интерактивный образовательный веб-сайт, который учит детей экологическому сознанию через игры и учебную деятельность.

## 🌍 Особенности

- **Экокруг**: Узнайте о воде, мусоре, лесах, транспорте и энергии
- **Экорюкзак**: Откройте для себя исчезающих животных
- **Экологические игры**: Интерактивные игры о сортировке, энергии, лесах и транспорте
- **Экотермины**: Образовательные определения

## 🚀 Начало работы

1. Откройте файл `index.html` в вашем веб-браузере
2. Нажимайте на экокруги, чтобы узнать о каждой теме
3. Играйте в образовательные игры
