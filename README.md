# automatic-umbrella
Создай яркий интерактивный сайт для проекта «Экологические привычки будущего» для детей 4–7 лет. Добавь «Экокруг» с 5 экопривычками, мини-игры, экорюкзак с животными из Красной книги, простые объяснения экотерминов, экологические задания, анимации, достижения и дружелюбный детский дизайн.
